# GIAAT Web — Angular 17

## Setup
```bash
npm install
npm start         # http://localhost:4200
npm run build     # Producción en dist/
```

## Agregar imágenes
Colocar en `src/assets/images/`:
- `specialties/geotecnia.jpg` `ingenieria.jpg` `arquitectura.jpg` `ambiental.jpg` `topografia.jpg`
- `projects/` — una imagen por proyecto (nombres en projects.data.ts)
